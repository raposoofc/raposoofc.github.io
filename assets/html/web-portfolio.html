Sobre Minha Jornada

Sou Rodolpho Rapôso, um desenvolvedor web front-end apaixonado por criar experiências digitais que combinam estética e funcionalidade. Minha jornada na programação começou com a curiosidade em desvendar como a web funciona e evoluiu para uma paixão por construir interfaces que são intuitivas e visualmente atraentes.
